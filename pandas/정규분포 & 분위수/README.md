예를 들어 남자와 여자 , 성공 실패등 2개의 모집단인 경우이다.

p를 하나의 확률이라고 할때, 나머지 확률 q라고 하면, 

불연속 변수는 수치 사이에 중간값이 없다.

__1) p + q = 1__
 
__2) p = q이면 p = q = 0.5__
