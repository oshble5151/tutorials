## grouby 
grouby는 그룹을 결정하는데 사용된다. 

grouby는 다음 3가지 프로세스를 실행가능하다.

**1. Splitting : 데이터를 몇가지 기준으로 나눔**

**2. Applying : 각 그룹에 대한 독립적인 함수 적용**

**3. Combining : 결과를 데이터 구조로 변환함**

## Splitting
Splitting 과정을 통해, 데이터를 간단하게 분할 할 수 있다.
