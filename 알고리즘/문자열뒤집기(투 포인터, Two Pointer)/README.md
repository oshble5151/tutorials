본 게시글에서는 문자열 뒤집기 문제를 투 포인터 알고리즘으로 풀어보고자 한다.

# 투 포인터(Two Pointer)

투 포인터는 오른쪽과 왼쪽의 포인터가 독자적으로 움직이는며 풀어가는 알고리즘이다.

즉 오른쪽 포인터를 이동하며 답을 찾으면 왼쪽 포인터를 이동시키는 알고리즘이며, 정답이 되지 않는 경우는 제외한다는 특징이 있다.
